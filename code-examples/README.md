# code-examples

**Sammelordner für Code-Stubs, Beispiele und Prototypen im Elysium-Ökosystem.**

Aktuell enthält dieser Ordner:

- `grok-launcher/` – Erster Rust + egui MVP Stub für den zentralen Grok Launcher

Weitere Beispiele (Python Agents, Mesh-Node-Skripte, Blockchain-Integrationen etc.) werden schrittweise hinzugefügt.

Jedes Beispiel ist bewusst als Einstiegspunkt gestaltet und verweist auf die detaillierten Dokumente in `docs/`.
